Untappd Integration
Tap a beer to view additional information about the beer from Untappd.

Login and authorize this application to view personalized information about the beer from Untappd. When logged in, your personal rating and number of checkins for the beer are displayed.
note: The login feature uses the OAuth standard from the Untappd API, which means this app never has access to or sees your credentials.

Tap the button to launch the Untappd app or Untappd website directly to the beer.

Filtering
The festival spans two days with a different list on each day. Additionally, the Bulls Head Backstage Pass beer list contains exclusive offerings available only to pass holders. Use the toggle buttons to filter to either day or the Backstage Pass.

Enter filter criteria at the top of the beer list. All fields are immediately filtered upon as you type. For example, type "IPA" and any beer name or style containing "IPA" will be found. The number of beers matching the current filter criteria is reflected in the number next to the filter input.

Shortcut
If your mobile device's browser is equipped with an "add to homescreen"-like feature, you can utilize it to gain a quick launch icon to this app.
